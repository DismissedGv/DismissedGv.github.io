Wratful Hedgehog

About

Wrathful Hedgehog was our first game project at FutureGames and it was made with C#/Unity together with a team assigned by FutureGames.

It's a game about deleting the devevelopers of the game and as you delete the crew, you also delete the game which is the end goal.

Info

Role: Gameplay Programmer

Team Size: 14

Time: 2 Months

Engine: Unity

Introduction

In this project I took the responsibility of programming the game's artificial intelligence since that was our second core mechanic of the game. We use github desktop as source control, miro for prototyping, jira for project management. I mainly put lot of focus on AI but I also worked making the map shrink, health, and UI. I also wanted to get a better understanding of unity AI since it was my first time using it.

Artificial Intelligence

This was my first time doing artificial intelligence in anything, therefore nothing too complicated was made here. I downloaded Unitys AI Navigation to be able to use a Navmesh for the ground surface and nav mesh agent so the enemies know where to walk and avoid obstacles.

What I learned

Working on this project with someone other than myself taught me a lot. I learned that it is important to not assume things, even if it seems obvious to you. We all work differently. After the project we also did code reviews. This was edducational for me to see another persons coding style when it comes to game development.
