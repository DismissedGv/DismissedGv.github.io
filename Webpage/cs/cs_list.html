C# List

In C#, a List is a generic collection class provided by the .NET framework that is part of the System.Collections.Generic namespace. It is used to store and manage a collection of objects of a specific type. Unlike arrays, List is dynamic, meaning it can grow and shrink in size as needed.


Key Features of List:

1. Generic Type:

  • List<T>: uses generics, where T represents the type of elements stored in the list. It ensures type safety.
  • Example: List<int>, List<string>, or even custom classes like List<Employee>.

2. Dynamic Sizing:

  • Lists automatically resize as elements are added or removed. You do not need to define the size beforehand.

3. Indexed Access:

  • You can access elements of the list using an index, just like an array.

4. Built-in Methods:

  • Lists provide several methods to manipulate data, such as Add(), Remove(), Insert(), Sort(), Find(), and many more.

5. Zero-based Indexing:

  • Like arrays, Lists also start indexing at 0.
